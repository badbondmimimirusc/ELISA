Alle Befehle


Allgemein

  • Hotword: (Hey / Hallo) Elisa
  • Öffne: «Öffne/zeige/starte Name», «Öffne Name in einem neuen Fenster»
  • Schliesse: «Schliesse/beende/stoppe Name», «Schliesse dieses Fenster/Programm»
  • Maximiere / minimiere: «Maximiere/minimiere dieses Fenster/dieses Programm/Name»
  • Text bearbeiten: «Wähle alles aus», «Kopiere», «Schneide aus», «Füge ein»
  • Computer sperren: «Sperre den Computer»
  • Bildschirmfoto: «Mache ein Bildschirmfoto/einen Screenshot»
  • Taste drücken: «Drücke Enter/Tab/Escape»

Word, Excel und Powerpoint

  • Neue Datei: «Erstelle eine neue Datei/neues Dokument»
  • Datei öffnen: «Öffne eine Datei/ein Dokument»
  • Datei speichern: «Speichere/sichere die Datei/das Dokument»
  • Datei speichern unter: «Speichere/sichere die Datei/das Dokument als/unter»
  • Schriftgrösse ändern: «Setze/stelle/mache die Schriftgrösse auf/zu Zahl», «Vergrössere/Verkleinere den Text.»
  • Texteigenschaften ändern: «Mache den Text (nicht) fett/kursiv/unterstrichen»
  • Farben: automatisch/schwarz/grau/weiss/blau/grün/pink/rot/türkis/violett/gelb/hellgrün/dunkelblau/dunkelrot/dunkelgelb
  • Text färben: «Färbe/Mache den Text Farbe»

Word

  • Objekt einfügen: «Füge ein/e (neue/s) Bild/Tabelle/Seitenzahl ein»
  • Text formatieren: «Formatiere den Text als Titel/Überschrift/Untertitel/Zitat/Liste/Aufzählung/Nummerierung/normal»
  • Texthintergrund färben: «Markiere den Text Farbe», «Färbe/Mache den Hintergrund Farbe»

Excel

  • Zelle färben: «Mache/Färbe/Markiere/Fülle die Zelle Farbe»

PowerPoint

  • Neue Folie: «Erstelle eine neue Folie»
  • Folie wechseln: «Gehe zur nächsten/vorherigen/ersten/letzten Folie/Seite»
  • Folie wechseln: «Gehe zur Folie/Seite Zahl», «Gehe zur X-ten Folie»
  • Bildschirmpräsentation: «Starte/Stoppe die Bildschirmpräsentation.»
